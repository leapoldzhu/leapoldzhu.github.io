个人工具链

记录个人工具链，帮助自己养成良好的学习习惯。时间、项目管理主要用到 Trello、滴答清单；笔记主要用到 OneNote、印象笔记；思维导图主要用到 scapple / cmap。其中印象笔记用作总资料库。

[TOC]

系统学习

场景 -

  1. 通过 慕课 进行课程学习
  2. 通过 书本 进行系统学习

工具链 -

Trello — 滴答清单 — OneNote — 印象笔记 — scapple / cmap

  • Trello

    对课程 / 书本进行拆分并记录进度

  • 滴答清单

    根据拆分的任务安排具体时间，并跟踪进度

  • OneNote

    记录课程内容笔记

  • 印象笔记

    将 OneNote 内容转换为 PDF 进行统一存储

  • Scapple / cmap

    画出相应的 思维导图 / 概念图

零碎学习

场景 -

  1. 突然学习 的知识点
  2. 微信 / 博客等 文章阅读

工具链 -

feedly — pocket — 滴答清单 — 印象笔记 — scapple / cmap

  • feedly

    RSS 订阅，每天定时检查

  • pocket

    通过 feedly 找出好文章保存到 pocket 中

  • 滴答清单

    根据需要进行记录并在之后阅读

  • 印象笔记

    以 知识卡片 形式记录

  • Scapple / cmap

    每 天 / 周 总结画出相应的 思维导图 / 概念图

  2. 个人 项目进行跟踪
